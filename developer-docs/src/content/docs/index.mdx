## Overview

These docs cover everything you need to build bots and integrations for Bedrock Chat. Topics include authentication, the REST API, webhooks, real-time events, and deployment.

### What you can build

- **Automated moderation** — filter content, enforce rules, and manage members programmatically
- **Custom commands** — respond to slash commands with dynamic data, lookups, or actions
- **Scheduled messages** — post reminders, announcements, or digests on a recurring basis
- **Third-party integrations** — connect Bedrock Chat to external services like GitHub, Jira, or your own internal tools
- **Interactive experiences** — build trivia games, polls, onboarding flows, or anything that responds to user input

### Privacy

<PrivacyNote>
These docs collect no user data — no cookies, no analytics, no tracking. The Bedrock Chat API enforces the same privacy standards. Bots cannot access private user data beyond what users explicitly share in channels the bot has permission to read.
</PrivacyNote>
