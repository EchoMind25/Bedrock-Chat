## Prerequisites

Before building a bot, make sure you have the following:

- **A Bedrock Chat account.** Sign up at the Bedrock Chat website if you don't have one.
- **Developer role approval.** A quick application inside Bedrock Chat. Covered in detail on the [Becoming a Developer](/getting-started/becoming-a-developer) page.
- **Node.js 18+** installed on your machine (for programmatic bots; not required for webhook-only bots).
- **A text editor.** VS Code, Sublime Text, or anything you prefer.
- **A terminal.** The built-in terminal on macOS, Linux, or Windows works fine.
- **Internet access.**

Your first webhook bot requires no coding. If you want to build interactive bots later, basic JavaScript or TypeScript knowledge helps but isn't required upfront.

## Getting Help

If you run into problems:

- **Check the troubleshooting page.** Most common issues are covered there.
- **Ask in the Bedrock Community server.** Other developers are active and helpful.

## A Note for Parents

Bedrock Chat enforces strict privacy controls. Bots cannot access private messages, email addresses, or personal information. All bots are reviewed by a human before they can be published publicly.

Your child will need an email address for their Bedrock Chat account and the developer application. A family email address works fine.
